# Roadmap

Building on: `yfinance`, `numpy-financial`, Streamlit

## Sections

### 1. Baseline DCF
A working discounted cash flow calculator that fetches real data (AAPL, TSLA) and outputs a raw value.

### 2. Educational Overlay
Add the "Teacher Layer": Tooltips explaining each input, "Step-by-Step" tab, and helper text ("Why does this matter?").

### 3. Scenario Playground
Interactive heatmaps and sensitivity analysis ("What if growth is 2% lower?") to show valuation ranges.

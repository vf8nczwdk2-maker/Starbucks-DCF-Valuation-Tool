# COF/Brex Merger Model

## The Deal
Capital One (COF) acquires Brex for **$5.15 Billion** in Jan 2026.
Structure: 50% Cash, 50% Stock.

## Key Assumptions
- **COF Price**: $235
- **Brex Revenue**: $1B (2025E), Growing 40% YoY.
- **Synergies**: Revenue synergies from $13B deposits (Net Interest Margin on these deposits).
- **Fees**: $50M.

## Output Requirements
1.  **Sources & Uses**: Balance the checkbook.
2.  **PPA**: Calculate Goodwill.
3.  **Pro Forma Impact**: Will this add to EPS (Accretive) or subtract (Dilutive)?
4.  **Sensitivity**: What if COF stock drops? What if Brex grows slower?

## Strategic Rationale
- **B2B Payments**: Instant entry into corporate spend management.
- **AI Integration**: Leveraging Brex's tech stack for COF's commercial clients.
